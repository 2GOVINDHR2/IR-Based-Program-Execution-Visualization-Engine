----------------------------------

----------------------------------

Access array element:

value = arr[mid]

----------------------------------

Check if value == target
If true → goto L1

----------------------------------

----------------------------------

----------------------------------

L3: SEARCH LEFT

----------------------------------

L5: NOT FOUND
